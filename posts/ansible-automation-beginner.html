Ansible Automation for Beginners

Learn how to automate server configuration and deployment using Ansible playbooks and roles.

  1. Cài đặt Ansible: 
    sudo apt update && sudo apt install ansible -y
  2. Tạo file hosts chứa danh sách server: 
    [web]
192.168.1.10
192.168.1.11
  3. Viết playbook đơn giản setup.yml: 
    - hosts: web
  become: yes
  tasks:
    - name: Cài nginx
      apt:
        name: nginx
        state: present
  4. Chạy playbook: 
    ansible-playbook -i hosts setup.yml
  5. Tìm hiểu thêm về roles, variables, handlers để tự động hóa phức tạp hơn.

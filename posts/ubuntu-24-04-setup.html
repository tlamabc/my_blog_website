Ubuntu 24.04 LTS Setup

Step-by-step guide to install Ubuntu 24.04 with LAMP stack, SSH, and basic security.

  1. Download Ubuntu 24.04 ISO: Get it from the official Ubuntu website.
  2. Create Bootable USB: Use Rufus (Windows) hoặc balenaEtcher (Linux/Mac).
  3. Install Ubuntu: Boot từ USB và làm theo hướng dẫn cài đặt.
  4. Update hệ thống: 
    sudo apt update && sudo apt upgrade -y
  5. Cài đặt LAMP stack: 
    sudo apt install apache2 mysql-server php libapache2-mod-php php-mysql -y
sudo systemctl enable apache2
sudo systemctl enable mysql
  6. Cài đặt SSH: 
    sudo apt install openssh-server -y
  7. Bảo mật cơ bản:
    • Đổi port SSH, tắt root login trong /etc/ssh/sshd_config
    • Cài đặt UFW: 
      sudo apt install ufw -y
sudo ufw allow OpenSSH
sudo ufw enable
  8. Hoàn tất! Truy cập http://your_server_ip để kiểm tra Apache.

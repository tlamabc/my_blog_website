Windows Server 2022 AD DS

Configure Active Directory Domain Services on Windows Server 2022.

  1. Cài đặt Windows Server 2022 và cập nhật hệ thống.
  2. Mở Server Manager > Add roles and features.
  3. Chọn Active Directory Domain Services và cài đặt.
  4. Chạy Promote this server to a domain controller sau khi cài xong.
  5. Chọn Add a new forest, nhập tên domain (VD: lam.local).
  6. Thiết lập password cho Directory Services Restore Mode.
  7. Chờ server tự động reboot và hoàn tất cấu hình.
  8. Kiểm tra bằng Active Directory Users and Computers.
